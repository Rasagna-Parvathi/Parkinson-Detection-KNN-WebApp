# Parkinson Disease Detection using Machine Learning

This project is a Machine Learning based web application that predicts whether a person has Parkinson's disease using voice measurement data.

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML/CSS

## Machine Learning Model
The system uses the K-Nearest Neighbors (KNN) algorithm to classify whether a patient has Parkinson’s disease based on biomedical voice measurements.

## Project Features
- Data preprocessing and scaling
- Machine learning model training
- Web interface using Flask
- Real-time prediction through web form

## How to Run the Project

1. Clone the repository
2. Install required libraries
